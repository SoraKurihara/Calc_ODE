# ODE_Solver

ODE(常微分方程式)を解くためのモジュールです．

# Features

numpy配列を用いることで，数値計算を高速におこなうことができるモジュールとなっています．
また，数値計算に用いる方法（現在はEuler法とRunge-Kutta法のみ）を選ぶことができます．

# Requirement

* numpy 1.24.1

# Usage

```bash
git clone https://github.com/SoraKurihara/Calc_ODE
cd Calc_ODE
python demo.py
```
